## Overview

Photo Mechanic Advanced is an ultra-fast photo browsing and culling application from Camera Bits designed for photojournalists, sports photographers, wedding photographers, and event shooters who need to review, rate, caption, and deliver hundreds or thousands of images as quickly as possible. It is not a photo editor — it is the fastest image browser available, purpose-built for the post-shoot workflow where speed determines whether images meet publication deadlines.

Instant browsing displays full-resolution previews of RAW files immediately without waiting for rendering, using embedded JPEG previews that provide instant visual evaluation while full-resolution data loads in the background. The browsing speed makes reviewing thousands of images from a shoot practical rather than tedious. No other application matches Photo Mechanic's image display speed.

Culling workflow with ratings, color labels, and tags enables rapid selection of keepers from large shoots using keyboard shortcuts that never require taking hands off the keyboard. The rating workflow is designed for speed — a single keystroke rates an image and advances to the next. Filtered views show only selected images for secondary review.

Captioning and metadata editing applies IPTC caption, keywords, copyright, and other metadata to images individually or in batches with code replacements that insert variable data — date, camera, lens, location — automatically. The captioning tools handle the metadata requirements of wire services and publication workflows.

For professional photographers who need to process large shoots fast enough to meet deadlines, Photo Mechanic Advanced is the browsing tool that makes post-shoot image handling as fast as physically possible.

## Performance

Photo Mechanic balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
